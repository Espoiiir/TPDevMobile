# TPFinalMobile

Groupe: Evan PERNOT et Julian MORALES

URL GIT : https://github.com/Espoiiir/TPDevMobile

Application Flutter utilisant l'API publique Raider.IO:
`https://raider.io/api`.

Modules:

- Guildes raid: appels `/api/v1/guilds/profile` et `/api/v1/raiding/raid-rankings`, recherche par région/royaume/nom et leaderboard top 10.
- Recherche personnage: appel `/api/v1/characters/profile`, profil détaillé avec score Mythic+, rangs, progression raid, équipement, ilvl et meilleures clés.


Ce qui nous a challengés : 
L'implémentation de l'API et le design de l'application



Ce dont on est fier :
L'ajout des boutons de redirections vers la page web raider.io.
Le classement des 10 premières guildes monde, et le classement régional + la page du personnage
